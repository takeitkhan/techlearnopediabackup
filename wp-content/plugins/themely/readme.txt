== Description ==

Beautiful, secure and easy-to-use WordPress themes from the most talented creators around the world. Each theme has passed a manual quality & security review by our team. Find, preview & install the perfect theme for your next project!


== Installation ==

1. Log into your Admin Dashboard.
2. Navigate to the Plugins page.
3. Click on Add New button at the top of the page.
4. Click on Upload Plugin button at the top of the page.
5. Click on Choose File and select the plugin's zip file.
6. Click on Install Now button.
7. Navigate to the Themes page which has now appeared in the left navigation.
8. That's it, you're done!
